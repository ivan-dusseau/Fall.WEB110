HTML Elements

Tag Description Block, Inline, N/A Containter Or Empty
<!-- --> Annotations to HTML, invisible unless inspecting the element. n/a container
<!DOCTYPE> States the type of document to browsers and other developers. Allows better parsing of code. n/a empty
<a> Creates a hyperlink for in line of text, linking between documents and websites. inline container
<article> Creates a standalone block of content. block container
<aside> Creates an area of content not related to the core body of the page. block container
<body> Establishes a wrapper of a HTML document's body. n/a* container
<br/> Creates a single line break in a block of text. inline empty
<dd> Used to describe a term located in a description list. block container
<div> A general unspecified container. block container
<dl> Creates a list with terms and their descriptions. block container
<dt> Used to define the term from a description list. block container
<em> Adds emphasis to a line of text. Indicated by bolding and reader voice inflection. inline container
<fieldset> Groups related elements in a form. block container
<figcaption> Creates a caption for a figure. block container
<figure> Used to specify self-contained content, like illustrations, diagrams, photos, code listings, etc. block container
<footer> Creates a footer section, usually at the bottom of the page. Includes author name, copyright information, and contact information. block container
<form> Creates a form of empty fields for user input. block container
<h1> to <h6> Creates a sections of an html document with varying priority. Lower numbers indicate higher importance. block container
<head> Contains all of the information that needs to be pre-loaded fir a page. n/a container
<header> Creates a section for introductory content for its nearest parent sectioning element. block container
<html> Tells the browser that the document is in HTML, and contains every other element except !DOCTYPE. n/a container
<iframe> Imbeds another document into the page, such as an image. inline container
<img> Links to an image. inline empty
<input> Defines an area for user input. inline empty
<label> Creates a label for items in an interface. inline container
<li> Gives meaning to a list item. block container
<link/> Used for external style sheets to link an external document to the current one. n/a empty
<meta/> Used for web crawlers to identify features of the page. n/a empty
<nav> Contains an unordered list of navigational links, reserved for internal navigation. block container
<p> Creates a paragraph. block container
<script> Contains or points to external scripting. inline container
<section> Creates a section in a document. blok container
<strong> Makes text bold to identify important text. inline container
<style> Indicates in page CSS styling. n/a container
<table> Creates an HTML table with rows and headings. block container
<td> Contains and creates a standard cell from an HTML table. n/a container
<th> Contains and creates a header cell from an HTML table. n/a container
<title> Title of the webpage, displayed by the browser. n/a container
<tr> Contains and creates a row for an HTML table. n/a container
<ul> Creates an unordered list with bullet points. block container
